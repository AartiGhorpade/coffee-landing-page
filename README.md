# ☕ Coffee Landing Page

## 📌 Overview

This project is a modern and responsive coffee shop landing page developed by converting a Figma design into a functional web application. The goal of this project was to practice translating UI/UX designs into clean, reusable, and responsive React components.

---

## ✨ Features

* Fully responsive design (mobile, tablet, desktop)
* Pixel-perfect UI based on Figma design
* Clean and reusable component structure
* Modern UI/UX design

---

## 🛠 Tech Stack

* Frontend: React.js
* Build Tool: Vite
* Styling: Tailwind CSS

---

## ⚙️ How It Works

The application is structured using reusable React components such as:

* Navbar
* Hero Section

Each section is designed based on the Figma layout and styled using Tailwind CSS. The layout adapts to different screen sizes using responsive design techniques.
